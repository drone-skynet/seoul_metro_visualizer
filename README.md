## 실행 커맨드
streamlit run app.py

## 지하철 노선 정보
1호선 ~ 9호선, 수인-분당선, 우이신설선

## 타임라인
시간대별 데이터를 활용해 출퇴근 시간 및 교통량 변화가 뚜렷히 보이도록 시각화

## 사용 라이브러리
* <b>Pandas</b>: 데이터 분석
* <b>Numpy</b>: 행렬 연산
* <b>Pydeck</b>: 대규모 지리공간 데이터 시각화
* <b>Altair</b>: 선언적(declarative) 데이터 시각화

## 데이터 소스
서울 열린데이터 광장(https://data.seoul.go.kr)
